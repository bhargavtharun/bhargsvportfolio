Projects

Autonomous Medical Robot

Developed an autonomous medical robot integrating advanced features like object detection, face recognition, SLAM (Simultaneous Localization and Mapping), and precise pose estimation, designed for efficient operation in complex medical environments.

This project involved deep learning models for real-time object identification, facial recognition for secure access, and advanced navigation algorithms to maneuver through hospital corridors. It utilized ROS (Robot Operating System) for modular development and integration of various sensor data.

ETL Data Pipeline with Azure & Snowflake

Engineered robust and scalable ETL (Extract, Transform, Load) data pipelines leveraging Azure services and Snowflake for data warehousing.

Built scalable pipelines using Azure Data Factory and Azure Databricks, integrated data sources from various APIs and databases, and implemented data quality checks. Visualized dashboards were created using Power BI to provide critical business insights.

Kubernetes Cluster Automation (GCP)

Automated the deployment and management of Kubernetes clusters on Google Cloud Platform (GCP) using Infrastructure as Code (IaC) principles.

Implemented CI/CD pipelines with Jenkins and GitLab CI for seamless application deployments. Utilized Terraform for infrastructure provisioning and ensured high availability and fault tolerance through robust cluster configurations.
